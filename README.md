# CSharp_WebApp_BookStore


 1. The purpose of the application
  -----------------------


This project is aimed to develop an online bookstore web application at the Visual C# and .Net Framework environment. This app named BookTree. It gives user instant access to books list and provides books information; book image, book title, price, quantity, book description, year for book published. In addition, the user can search for the desired book by keyword like book author or book title. To order the book, the user needs to register their account. User can keep their book list at their shopping cart, and the cart represent userâ€™s book list when next login. While the user checks out books in their shopping cart and enter the shipping information they want, they can complete their order. User can check their order placed at their account page, modify the order details, and cancel their order as well before delivery.
